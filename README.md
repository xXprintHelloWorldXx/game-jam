# game-jam-template
Students will use this template to start their game jam projects.

1. Team leader forks a copy of the repository for personal use.
2. Team leader opens repository settings, goes to collaborators tab, and invites teammates to project.
3. Teammates accept invite in GitHub notifications.
4. All teammates can now clone repository to GitHub Desktop and open project in Gamemaker.
-Remember to create new branches before opening and editing the project in Gamemaker.
-When the feature you are working on is done, create a pull request to merge it into the main branch.
-Remember to back up your work often.
